# 🦖 Dino Game Clone

Um clone do famoso **jogo do dinossauro do Google Chrome**, desenvolvido utilizando a biblioteca **Phaser** como trabalho final da disciplina de LABORATÓRIO DE PROGRAMAÇÃO PARA WEB.

## 🎮 Sobre o Projeto

Este projeto recria a experiência clássica do jogo offline do Google Chrome, onde o jogador controla um dinossauro que deve desviar de obstáculos para alcançar a maior pontuação possível.

O objetivo principal deste projeto foi explorar conceitos de desenvolvimento de jogos 2D utilizando o framework Phaser, incluindo:

- Sistema de física e colisões
- Animações de sprites
- Geração procedural de obstáculos
- Controle de pontuação
- Gerenciamento de cenas
- Manipulação de eventos de entrada do usuário

## 🛠️ Tecnologias Utilizadas

- JavaScript (ES6+)
- Phaser 3
- HTML5
- CSS3
